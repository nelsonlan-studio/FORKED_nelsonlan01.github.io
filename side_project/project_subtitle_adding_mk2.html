Premiere Pro Subtitle Editing Tool MK.II

What is Subtitle Editing Tool MK.II?

Premiere Pro Subtitle Editing Tool MK.II is

  • ● Premiere Pro Video subtitle adding tool MK.II is 2nd of tool that Nelson developed for repetitive, and a lot of school video editing.
  • ● This tool replaced input subtitles directly into software nor easily messed up `.SRT` file.

  • Speeding up formatting and finalizing subtitles greatly, there is a minor problem that if the line is too long that needs manually fix.

Why develop MK.II? and Aim

  • ● MK1 Version fixed timestamp was not user-friendly for long videos.
  • ● MK1 Version is on CML run environment not on GUI.

Technologies / Skill Used

  • ● Unity 2021 (UI Element)
  • ● C# Code, with ChatGPT

MK.II new features

Introduction of timestamp function
  • ● Not in fixed spacing of 5 secs.
  • ● You can start video and tool together to co-editing your work

Operation System

  • ● Windows 10 above, or;
  • ● Unity engine-game-supported platform
